Choose Server Configuration

You can choose to configure your server as either a DHCP server or as a BOOTP relay agent.

The DHCP server manages the IP address space of networks connected to the server.

The BOOTP relay agent can receive requests for configuration from a client in a network not directly connected to a DHCP server and forward the request to a DHCP server.
