Delete Address

The Delete Address dialog box lets you delete one or more addresses from the selected DHCP network. The results of this dialog are analogous to those generated by the SolarisTM pntadm -D -y command.

Click OK to remove the selected addresses from the network table.

Click Delete From Hosts Table to also remove the entries associated with these addresses from the hosts table.
