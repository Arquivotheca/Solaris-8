Unconfigure BOOTP Service

Click OK to unconfigure the BOOTP relay service for this server. The server will cease to relay requests for DHCP configuration. To restore BOOTP service, you must reconfigure the server using the BOOTP configuration dialog box.

Use Stop or Disable from the Service menu to halt or disable DHCP service.
