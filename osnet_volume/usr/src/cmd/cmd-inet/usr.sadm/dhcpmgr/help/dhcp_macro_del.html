Delete Macro

Click OK to remove the macro from the DHCP database.

Notify DHCP Server of Change: Check the box if you want the DHCP server to reload the dhcptab after the changed macro information is written. Normally, you want the server to reload the dhcptab so that clients start receiving the new configuration data immediately. However, in some cases, you may want to delay this reload. For example, you may need to make several changes in order for configurations to be correct.
